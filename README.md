# The Imperial Guest House — Website

Premium boutique guest house website built with vanilla HTML, CSS, and JavaScript.

## Features
- 🌗 Dual theme (Light / Dark) with localStorage persistence
- 💬 WhatsApp booking integration
- 📱 Fully responsive design
- ✨ Hotel Seven7-inspired hover effects & animations
- 🖼️ Photo gallery with lightbox
- 📍 Google Maps integration

## Pages
- **Home** — Hero, booking bar, rooms, amenities, gallery, testimonials, location
- **About** — Story, values, features, stats
- **Rooms** — Standard, Deluxe, Premium Suite with comparisons
- **Amenities** — WiFi, parking, room service, and more
- **Gallery** — 6-image grid with lightbox viewer
- **Location** — Map, contact info, directions

## Tech Stack
- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript
- Google Fonts (Cormorant Garamond + DM Sans)

## Deploy

### Vercel
1. Push to GitHub
2. Import in [vercel.com](https://vercel.com)
3. Framework: **Other** → Deploy

### GitHub Pages
1. Go to repo Settings → Pages
2. Source: Deploy from branch → `main` → `/ (root)`
3. Save
